## Getting Started
```bash
npm install
npm run dev
```
## Deployment
```bash
npm run build
npm run preview
```
## Troubleshooting
If you encounter any issues:

Delete node_modules folder and package-lock.json

Run npm install again

Ensure you're using Node.js 20.19.0
